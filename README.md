# Customer Segmentation System

A Machine Learning based Customer Segmentation web application developed using Python, Flask, and K-Means Clustering to group customers based on their purchasing behavior and annual income.

## Technologies Used
- Python
- Flask
- HTML/CSS
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Features
- Customer Group Classification
- K-Means Clustering Algorithm
- Data Visualization
- Customer Spending Analysis
- Interactive Web Interface

## Project Modules
- Data Preprocessing
- Feature Scaling
- Customer Clustering
- Model Training
- Result Visualization

## Dataset Used
- Mall Customers Dataset

## Future Enhancements
- Advanced Customer Analytics
- Real-time Customer Insights
- Interactive Dashboard
- AI-based Recommendation System
